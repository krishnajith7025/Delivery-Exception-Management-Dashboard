# Delivery Exception Management Dashboard

## Project Overview

The Delivery Exception Management Dashboard is a frontend web application developed using HTML, CSS, and JavaScript. It allows users to report delivery issues, view reported exceptions, track their status, filter records, and manage delivery exceptions through a simple and user-friendly dashboard interface.

## Features

- Report new delivery issues
- Required field validation
- Dynamic dashboard table
- Resolve issues
- Delete issues with confirmation
- Filter by Issue Type
- Filter by Status
- Open and Resolved issue counters
- High Priority issue highlighting

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.

## Project Files

- index.html – Structure of the dashboard
- style.css – Styling and layout
- script.js – Dashboard functionality
